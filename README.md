#OmniBot: Multi-Modal Local AI Co-Pilot for Web Applications
∗ Developed a reusable React AI widget integrated with Ollama for 100% local, privacy-first multi-modal inference using Llama3 and Llava models.
∗ Engineered a Visual Context Engine using WebRTC and getDisplayMedia to provide the LLM with real-time screen awareness and continuous visual reasoning.
∗ Implemented an advanced Function Calling (Tool Use) system allowing the AI to autonomously trigger client-side actions and manipulate the host website’s DOM.
∗ Created a custom "Circle to Search" feature using html-to-image and Canvas API to capture and analyze specific UI regions via vision-language models.
∗ Optimized user experience with a physics-based draggable UI using Framer Motion and global keyboard shortcuts for seamless workflow integration.
